To use your own logo/icon:

1. Create a 512x512 PNG image
2. Save as 'source-icon.png' in project root
3. Install sharp: npm install --save-dev sharp
4. Run: npm run generate-icons
5. Rebuild: npm run build

Or use online tool:
   https://realfavicongenerator.net/
